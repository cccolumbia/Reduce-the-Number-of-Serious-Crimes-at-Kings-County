# Reduce-the-Number-of-Serious-Crimes-at-Kings-County

## Techniques used
Basic EDA, data transformation, Correlation test and improvement, Quantiles, Interaction test and improvement, Tukey multiple comparison, ANOVA, Stepwise regression, Linear regression, Boxplot, Histogram.

## Data Source  
County demographic information (CDI) data set from Applied Linear Statistical Models, 5th edition, by Kutner, Nachtsheim, Neter, and Li., Appendix C2 (APPENC02.txt).   
The project selects 250 counties randomly from the 440 counties included in the whole data set.

## Conclusions

From all analysis above, factors should be responsible for large number of serious crimes in Kings county are:     

large population (high Population land ration, large Total population, more Number of hospital beds), large proportion of population aged 18-34(greater Percent of population aged 18-34, large Number of active physicians, high percent of unemployment ), high poverty level(higher Percent below poverty level, less Total personal income), low education level(high Percent of high school graduates, low Percent of bachelorâ€™s degree).      

To reduce number of serious crimes in Kings county, our suggestions are :   

(1) Control population increase, implement family planning policies.      

(2) Solve poverty problem, encourage people,especially young people to work inside or outside Kings county.        

(3) Develop local educational institution and support inhabitants to chase for higher level degree outside Kings county.
